# CSBNTCGTW
## 简介：
> 此项目是2018年8月份写的一个关于在nuke里面创建镜头任务的插件，可以自动判断拖入的序列帧然后链接资产管理软件创建镜头任务。主要提供给渲染主管使用。
## 使用条件：
>* Nuke10
## 用法：
>* 1.首先把文件夹复制到C:\Users\<username>\.nuke文件夹下面
>* 2.然后把menu.py文件复制到C:\Users\<username>\.nuke文件夹下
